## Getting Started
Install node.js:
```
$ npm install node.js
```
Install mongo db:
```
$ npm install mongodb.js
```
Install nodemon
```
$ npm install nodemon
```
Install Express
```
$ npm install express
```
Install standard
```
$ npm install standard
```
Install cors
```
$ npm install cors
```
Install mongoose
```
$ npm install mongoose
```
Install dotenv
```
$ npm install dotenv
```
Install bcrypt
```
$ npm install bcrypt
```
Install jsonWebToken
```
$ npm install jsonwebtoken
```
Run in dev mode:
```
$ npm run dev
```
